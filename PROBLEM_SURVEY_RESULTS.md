## Analysis Methodology

### Mean Score Calculation
```
Mean Score = (5×SA + 4×A + 3×N + 2×D + 1×SD) / Total Responses
Where:
SA = Strongly Agree count
A = Agree count
N = Neutral count
D = Disagree count
SD = Strongly Disagree count
```

### Agreement Percentage Calculation
```
Agreement % = (Strongly Agree + Agree) / Total Responses × 100
```

### Priority Ranking
Problems are ranked by:
1. Mean Score (primary)
2. Agreement Percentage (secondary)
3. Frequency of mention in open-ended responses (tertiary)
